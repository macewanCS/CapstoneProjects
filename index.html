Edmonton Public Library Digital Wall Interactive Game

Candy Pang <candy.pang@macewan.ca>

2025-05-26

Description:
Collaborate with Edmonton Public Library to create an interactive game for the Milner Digital Wall. The project will follow the Agile methodology. The game will be available to the general public between 5-year-old and 100-year-old. The game design will satisfy a wide range of reading levels, languages, abilities, and expectations. The game's content can be educational, informational, or just for fun.
